NightSignal T3 Dose Response Detection

Adapting Snyder Lab's COVID detection methodology for thyroid dosing optimization

Methodology (from Alavi et al., Nature Medicine 2022)

  • Individual baseline comparison - compares each person to their own 28-day rolling median
  • Overnight RHR - uses 00:00-07:00 heart rate when truly at rest
  • Bidirectional detection - elevated RHR = possible hyperthyroid, depressed RHR = possible hypothyroid
  • State machine alerting - Green (normal) → Yellow (single deviation) → Red (2+ consecutive days)
